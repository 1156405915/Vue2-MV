# Vue2-MV

> 这是一个基于Vue2实现的网易云音乐MV的webapp。

[在线地址](https://mv.mrcxh.com/)

### 主页功能
- 查看搜索MV
- 观看MV
- 查看MV评论

## 安装运行（Build Setup）

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:5000
npm run dev

# build for production with minification
npm run build

```
